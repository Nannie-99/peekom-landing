# 빼꼼 인덱스 / 플러스 QA 체크리스트

수동 테스트용 회귀 점검 목록입니다. `npm run dev`로 앱을 실행한 뒤 확인하세요.

## 1. 설치 · OS 지원

- [ ] Windows 10/11에서 설치 및 실행 정상
- [ ] Windows 7/8/8.1에서 설치기가 차단 메시지를 표시함 (또는 실행 시 “Windows 10 이상” 안내 후 종료)
- [ ] 다운로드 페이지에 “Windows 10 이상 권장” 문구 표시

## 2. 상태 동기화 (제보 #2)

- [ ] 메인 패널에서 인덱스 추가 → 설정창이 **이미 열려 있어도** 새 탭이 즉시 나타남
- [ ] 트레이 메뉴 Add로 인덱스 추가 → 설정창 탭 개수 반영
- [ ] 설정창에서 인덱스 삭제 → 메인 손잡이 개수 즉시 반영
- [ ] 설정창 포커스만 다시 줘도 최신 인덱스 목록 반영
- [ ] 설정창에 미저장 변경이 있을 때 외부 변경 시 확인 대화상자 표시

## 3. 삭제 · 시작 프로그램 (제보 #3)

- [ ] “Windows 시작 시 자동 실행” 켠 뒤 제어판에서 앱 제거
- [ ] 재부팅 후 이상한 오류 창/텍스트가 나타나지 않음
- [ ] 작업 관리자 → 시작 프로그램에 빼꼼 관련 항목이 남지 않음
- [ ] 자동 실행 체크 해제 시 OS 등록이 실제로 해제됨 (체크박스와 OS 상태 일치)

## 4. 인덱스 경계값

- [ ] 인덱스 10개일 때 추가 버튼 비활성/안내 메시지
- [ ] 인덱스 1개일 때 삭제 시도 시 안내 메시지
- [ ] 삭제 후 안전한 탭으로 이동

## 5. 메모 · 자동 접힘

- [ ] 메모 입력 중(포커스) hover 자동 접힘 유예
- [ ] 서식바 드롭다운 열린 동안 자동 접힘 유예
- [ ] 이미지 넣기 미리보기 중 자동 접힘 유예
- [ ] 이미지 넣기 중 다른 인덱스 이동 시 확인 대화상자

## 6. 설정창 UX · 프리미엄 (v1.2.0)

- [ ] 무료: 표시 모니터 드롭다운 사용 가능 / 라벨 클릭 시 Plus 모달 없음
- [ ] 무료: 불투명도 슬라이더 100% 고정, 슬라이더 클릭 시 Plus 모달
- [ ] 무료: 백업 펼치기·보내기·복원 전부 차단, 🔒 Plus 배지가 토글과 겹치지 않음
- [ ] 무료: A−/A+/글꼴 select 클릭 시 Plus 모달 / 라벨 클릭 시 모달 없음
- [ ] 무료: 커스텀 테마 버튼 클릭 시 Plus 모달, 색 팔레트 미표시
- [ ] 무료: 인덱스 3개 후에도 + 버튼 표시 → 클릭 시 Plus 모달
- [ ] Plus: 위 기능 정상 해제 (10슬롯·백업·커스텀·불투명도 등)

## 7. 이미지 · 인덱스 (v1.2.0)

- [ ] 이미지: 선택 → 크기 조절(스테이징) → 넣기 → 메모 본문에 바로 삽입 (2차 슬라이더 없음)
- [ ] 무료: 이미지 크기 슬라이더 100% 고정 + 클릭 시 Plus 모달
- [ ] 설정 → 인덱스 탭: 첨부 이미지 썸네일 표시, 클릭 시 확대 보기
- [ ] 손잡이 우클릭 → 인덱스 추가/삭제 동작, 설정창 탭 동기화

## 8. 회귀 (기존 기능)

- [ ] 손잡이 클릭 10회 연속 → 깜빡임·점프 없음
- [ ] 메모 비율 1:1 / 1:2 / 2:1 전환
- [ ] 서식바 정렬·색·형광펜 클릭 드롭다운
- [ ] 이미지: 선택 → 조절 → 넣기 (메모 본문 인라인)
- [ ] JSON 백업보내기 / 불러오기
