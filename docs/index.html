자주 묻는 질문

빼꼼 인덱스와 빼꼼 플러스에 대해 자주 묻는 내용입니다.

빼꼼 인덱스와 빼꼼 플러스의 차이는 무엇인가요?
듀얼 모니터에서 어떻게 동작하나요?
빼꼼 플러스에서는 설정에서 힌트 노트를 표시할 모니터(타겟 모니터)를 직접 선택할 수 있습니다.
빼꼼 플러스 라이선스는 어떻게 인증하나요?
Lemon Squeezy를 통해 구매한 라이선스 키를 앱 최초 실행 시 입력하면 인증됩니다.
Windows 설치 시 파란 경고창이 뜹니다.
SmartScreen 경고는 서명되지 않은 앱에서 흔히 나타납니다. 설치 가이드를 참고해 [추가 정보] → [실행] 순서로 진행하세요.

도움말

빠르게 시작하는 방법입니다.

1. 설치

운영체제에 맞는 설치 파일을 다운로드한 뒤 실행합니다. Windows에서는 SmartScreen 경고가 나올 수 있습니다.

2. 손잡이 위치 조절

모니터 우측 가장자리의 손잡이를 드래그해 원하는 높이로 이동할 수 있습니다. 마우스를 올리면 힌트 노트가 슬라이드됩니다.

3. 빼꼼 플러스 전용 — Pin 모드

손잡이를 클릭하면 노트가 고정(Pin)되어 호버 없이도 계속 표시됩니다. 0.25초 호버 딜레이로 오작동을 방지합니다.
